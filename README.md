# Installation

Clone repository

> git clone https://github.com/Trosdan/GoEcommerce.git

Download dependencies with yarn

> yarn

Run the fake API

> npm i json-server -g

> json-server server.json

Run the project

> react-native run-android

or

> react-native run-ios
